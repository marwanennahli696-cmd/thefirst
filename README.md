# Photos des villes

Déposez ici une image par ville en respectant le nom de fichier
`<city_id>.{jpg,jpeg,png,webp}` — par exemple :

- `marrakech.jpg`
- `fes.jpg`
- `chefchaouen.jpg`
- `essaouira.jpg`
- `casablanca.jpg`
- `rabat.jpg`

Recommandations :
- format paysage (idéalement 1600 × 900 ou plus),
- l'image est automatiquement recadrée pour remplir la zone (cover),
- un dégradé sombre est ajouté en bas pour la lisibilité du nom.

À défaut, une illustration décorative est générée à partir de la couleur
`accent` définie dans `data/cities.json`.
